# Public assets

Drop your logo here as **`logo.png`** — the Nav and Footer components reference `/logo.png` and will pick it up automatically.

Optional extras:

- `app/icon.png` — favicon (Next.js auto-detects); 32×32+ PNG
- `app/apple-icon.png` — iOS home-screen icon; 180×180 PNG
- `public/og-image.png` — Open Graph share image; 1200×630

That's it. The brand mark you uploaded is the file to save here.
